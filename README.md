# 💎 Professional Pendant Designer 3D Pro

A powerful, feature-rich 3D pendant jewelry design application with photorealistic rendering and professional-grade customization options.

## ✨ Features

### 🎨 Design Options
- **12+ Base Shapes**: Circle, Oval, Heart, Teardrop, Square, Hexagon, Octagon, Rhombus, Star, Cross, Shield, Infinity
- **9 Wing Styles**: Dual Symmetric, Single Left/Right, Angel, Bird, Ornate, Butterfly, Dragon
- **7 Wing Surfaces**: Smooth, Feathered, Ornate, Geometric, Pavé Diamonds, Textured, Crystal

### 💎 Gemstone Customization
- **10 Gem Shapes**: Round Brilliant, Heart, Oval, Cushion, Emerald, Princess, Pear, Marquise, Asscher, Radiant
- **9 Preset Gem Types**: Diamond, Ruby, Sapphire, Emerald, Amethyst, Topaz, Aquamarine, Opal, Pearl
- Full control over size, depth, facets, sparkle, transparency, and iridescence

### 🖼️ Advanced Features
- **Custom Materials**: 8 standard metals + custom color layering system
- **Border Styles**: Simple, Beaded, Ornate, Rope, Twisted, Crown, Pavé with optional gems
- **6 Bail Styles**: Simple, Ornate, Heart, Decorative, Integrated Gem Loop, Hidden
- **Text Engraving**: Custom text with 4 font styles and adjustable depth
- **5 Overlay Types**: Cross, Floral, Celtic Knot, Filigree, Geometric
- **8 Chain Styles**: Simple Cable, Box, Curb, Rope, Figaro, Wheat, Snake, Ball

### 🎮 Professional Controls
- Precise dimension control (width, height, thickness, bevel)
- Material properties (glossiness, metallic intensity, reflectivity)
- Advanced lighting controls (ambient, directional, environment, shadows)
- Real-time preview with smooth rotation
- Touch-optimized for mobile devices

### 💾 Export Options
- **3D Models**: STL (3D printing), OBJ, GLTF formats
- **High-Resolution Images**: 4K PNG screenshots
- **Design Specifications**: Complete JSON with all measurements and settings
- **Save/Load**: Design persistence using localStorage

### 🎁 Preset Templates
- Angel Heart
- Crystal Wings
- Royal Gemstone
- Celtic Cross
- Dragon Pendant
- Vintage Locket

## 🚀 How to Use

1. **Open** `index.html` in a modern web browser (Chrome, Firefox, Edge recommended)
2. **Customize** your pendant using the control panel on the right
3. **View** your design in real-time 3D with auto-rotation
4. **Export** your design in multiple formats when ready

## 🎨 Controls

- **Mouse**: Left click + drag to rotate, scroll to zoom
- **Touch**: Single finger drag to rotate, pinch to zoom
- **Keyboard**: Use the UI controls in the right panel

## 🔧 Technical Details

- **Built with**: Three.js (3D graphics library)
- **Rendering**: WebGL with physically-based materials
- **Lighting**: Environment mapping + dynamic lights
- **Shadows**: PCF soft shadows for realism
- **Responsive**: Fully responsive design for all screen sizes

## 💡 Tips

1. **Best Results**: Use Chrome or Firefox for optimal performance
2. **High Quality Exports**: Export STL for 3D printing, PNG for presentations
3. **Experimentation**: Try different combinations using preset templates
4. **Performance**: Reduce gemstone facets if experiencing lag on mobile
5. **Realistic Designs**: Enable iridescence on gems for authentic sparkle
